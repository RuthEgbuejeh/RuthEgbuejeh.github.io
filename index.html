Hi, I'm Ruth Isioma Egbuejeh

As a Relationship Manager and Data Analyst at Providus Bank, I specialize in transforming financial data into actionable insights for business loan applicants. With three years of experience, I’ve curated and analyzed financial records for diverse clients from new to existing banking customers, enabling data-driven lending decisions.

My expertise spans advanced analytics tools and visualization techniques, which I leverage to uncover trends, assess risk, and streamline reporting processes. Below, you’ll find a showcase of my technical toolkit and projects highlighting my ability to turn raw data into strategic value. GITHUB

PYTHON

I use Python for end-to-end data analysis, from cleaning with Pandas to advanced modeling with Scikit-learn. Automation, visualization (Matplotlib/Seaborn), and API integrations are key strengths in my toolkit. I build scalable scripts for ETL pipelines and predictive analytics.

Check out the Prosper Loan Performance Analysis Project

SQL

I leverage SQL to efficiently extract, transform, and analyze large datasets, optimizing queries for performance and accuracy. My expertise includes complex joins, subqueries, and window functions to uncover actionable insights. I design relational databases and maintain data integrity through normalization and indexing. SQL is my go-to for robust data manipulation and backend integration.

Check out the SQL Murder mystery Project

PowerBi

I create interactive dashboards in Power BI that translate raw data into strategic insights using DAX and Power Query. My designs prioritize user-friendly visuals and real-time data connectivity. I implement row-level security and optimize performance for large datasets. Power BI enables me to deliver compelling data stories to stakeholders.

Checkout the Tata Group Data Visualization Project
