PYTHON PROJECT: CLEANING THE MOVIES DATASET

I cleaned a movies dataset using Python's pandas library, starting by importing the data and removing irrelevant columns like rating, votes, and runtime to focus on core attributes. Next, I eliminated duplicate movie titles by sorting the dataset and applying drop_duplicates(), reducing entries from 9,999 to 6,817 unique films. For the genre column, I split multi-genre entries using str.split() and retained only the first genre (e.g., converting "\nAnimation, Action" to "Animation"), then standardized formatting with str.strip(). Finally, I transformed the year notation by extracting start years with regex and split operations, converted them to integers using pd.to_numeric(), and isolated lead actors in the stars column by splitting on "Stars:" and removing director details.
